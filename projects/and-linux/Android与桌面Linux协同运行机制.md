# Android与桌面Linux协同运行机制

Android与桌面Linux协同运行机制的研究的当前阶段已完成。

该研究作为我的论文课题，论文提出机制如下：
一种 Android 与桌面 Linux 应用程序协同运行的机制。此机制利用 Change Root 技术和 VNC 技术实现桌面 Linux 应用程序在 Android 系统上的运行，并且通过 Binder 扩展机制将 Android 四大组件扩展至桌面 Linux 应用程序，使得桌面 Linux 应用程序能够使用 Android 四大组件，从而实现与 Android 应用程序的交互。

具体请见项目：
https://github.com/HsingPeng/ExtendBinderExperiment
